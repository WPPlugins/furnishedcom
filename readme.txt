== Description ==

The Furnished.com WordPress plugin creates a powerful search widget on your website that allows users to browse through thousands of furnished apartments, homes, condos, townhouses, and other rentals right from your website.

With this powerful furnished rental widget you can configure which city to show furnished rentals from. It is perfect for any geographical website.

Each listing contains beautiful photos, price information, and a truncated description.

Sort by price, square feet, when it was created, as well as filter by price, bedrooms, and bathrooms.

All [furnished apartments](http://furnished.com/ "furnished apartments") and rentals are provided exclusively through [Furnished.com](http://furnished.com/ "Furnished.com"). 

== Installation ==

1. Unzip and place the 'furnished-com' folder in your 'wp-content/plugins' directory
2. Click the 'Activate' link for Furnished.com on your Plugins page (in the WordPress administration page)

== Frequently Asked Questions ==

= The plugin is not showing listings from the proper city. =

Ensure that the attributes being passed into the short code are set properly.
